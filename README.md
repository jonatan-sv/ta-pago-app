Este é um projeto de aplicativo para as cadeiras de **Projeto de Interfaces para Dispositivos Móveis** e **Projeto Integrado 4**.

## ⚡ Rodando a aplicação

 **1.** Depois de clonar o repositório, instale as dependências:

```shell
npm install
```

 **2.** Após baixar tudo, rode a aplicação:

```shell
npm start
```

 **3.** Um Código QR vai aparecer no terminal, então basta escanear com o Expo GO.
